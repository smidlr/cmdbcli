=================
 Running cmdbcli
=================

Setup
-----

First, you need to create a virtual environment and activate it.

.. code:: bash

  $ mkdir cmdbcli && cd cmdbcli
  $ pip install virtualenv
  $ virtualenv venv
  $ . venv/bin/activate
  (venv)$ 

Next, install requirements.

.. code:: bash

  (venv)$ pip install -r requirements.txt
